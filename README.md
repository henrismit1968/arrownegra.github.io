# ARROWNEGRA REPOSITORY
# Versão Actual: 2.05

Instruções para a adição deste repo:


<p align="left">
  <ul>
    <li>Ir para o Kodi gestor de ficheiros.</li>
    <li>Clicar em "Adicionar fonte"</li>
    <li>O endereço para a fonte é <code>https://arrownegra.github.io</code> (Dar o nome de "ArrowNegra Repo").</li>
    <li>Ir para "Addons"</li>
    <li>Em Addons, instalar de um ficheiro zip. Quando perguntar pela localização, selecionar "ArrowNegra Repo", e instalar <a href="repository.arrownegra-2.05.zip">repository.arrownegra-2.05.zip</a>.</li>
    <li>Voltar para Addons,mas desta vez, selecionar "Instalar do repositório"</li>
    <li>Selecionar o "ArrowNegra Repository"</li>
    
                                       !!!Agora o resto fica á vossa escolha!!!
  </ul>
</p>

## Enjoy!
